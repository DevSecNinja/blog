**Problem:** When you upload your new Windows 10 ADK boot image to SCCM 2012 R2 SP1, the process ends with the error "The specified file is invalid. Only maximum compression type is supported". **Solution:** Download the newest Windows ADK for Windows 10, Version 1511 [here at Microsoft](https://msdn.microsoft.com/en-us/windows/hardware/dn913721(v=vs.8.5).aspx).
